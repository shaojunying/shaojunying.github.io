题目

逆序对：对于(i,j)(i, j)(i,j)，如果满足i<ji < ji<j，并且nums[i]>nums[j]nums[i] > nums[j]nums[i]>nums[j]， 则称(i,j)(i, j)(i,j)为数组中的一个逆序对。

解法

暴力解法

思路

两轮循环：

  • 第一轮循环遍历每个元素，假设当前遍历到元素iii
  • 第二轮循环找到iii可以和后面元素组成的逆序对个数。

代码实现

public class Solution {
    public int reversePairs(int[] nums) {
        // 暴力解法 ==> 对于每个元素 向后遍历 计算包含该元素的逆序对个数
        int res = 0;
        for (int i = 0; i < nums.length; i ++) {
            for (int j = i + 1; j < nums.length; j ++) {
                if (nums[i] > nums[j]) {
                    res ++;
                }
            }
        }
        return res;
    }
}

时间复杂度

O(N2)O(N^2)O(N2)

分治解法（基于归并排序）

思路

利用归并局部有序的特性。

求数组nums的[l, r]之间的逆序对数：

  1. 将数组均分为[l, m], [m + 1, r]两部分。
  2. 分别求出[l, m]和[m + 1, r]两部分的逆序对个数。
  3. 排序[l, m]和[m + 1, r]。
  4. 逐个遍历，从两个数组找最小值。如果最小值在左边则不做处理，如果最小值在右边，则该值和左边的剩余元素都可以构成逆序对。

代码实现

public class Solution {
    public int reversePairs(int[] nums) {
        // 分治算法
        // O(NlogN)
        return reversePairs(nums, 0, nums.length - 1);
    }
    /**
     * 返回当前序列中的逆序对数
     * 将该序列中升序排序
     */
    private int reversePairs(int[] nums, int l, int r) {
        // [l, r]
        // 元素个数小于等于1 不用做任何处理
        if (l >= r) {
            return 0;
        }

        // 有多个元素 分为两部分
        int m = l + (r - l) / 2;
        // [l, m] [m + 1, r]
        int left = reversePairs(nums, l, m);
        int right = reversePairs(nums, m + 1, r);
        int i = l, j = m + 1;
        int mid = 0;
        // 定义一个新数组 拷贝原数组
        int[] copiedNums = new int[r - l + 1];
        System.arraycopy(nums, l, copiedNums, 0, r - l + 1);
        int k = l;
        while (i <= m && j <= r) {
            if (copiedNums[i - l] > copiedNums[j - l]) {
                mid += m - i + 1;
                nums[k] = copiedNums[j - l];
                j ++;
            }else {
                nums[k] = copiedNums[i - l];
                i ++;
            }
            k ++;
        }
        // 如果i还没走完 继续拷贝
        System.arraycopy(copiedNums, i - l, nums, k, m - i + 1);
        // mid += (r - m) * (m - i + 1);
        return left + mid + right;
    }
    
    public static void main(String[] args) {
        int res = new Solution().reversePairs(new int[]{7,5,6,4});
        System.out.println(res);
    }

}

时间复杂度

O(N∗logN)O(N*logN)O(N∗logN)
