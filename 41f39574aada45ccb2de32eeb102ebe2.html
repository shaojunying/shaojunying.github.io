在go语言下引入依赖

问题描述

执行 go get 命令下载gin包一直提示超时。尝试了下面的命令也不管用

export GOPROXY=https://goproxy.io
go env -w GO111MODULE=on

解决办法

其实上面的 go env -w GO111MODULE=on 命令是打开go mod格式来管理包。在这种方式下，我们可以使用go mod的方式来管理包

初始化项目为go mod项目

go mod init 项目名

下载项目中需要的依赖

go mod tidy

结果如下图所示，可以正常下载项目的依赖

启动项目

之后运行下面指令就可以启动项目

go run main/main.go
