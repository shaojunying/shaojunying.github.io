Mac下运行docker-maven-plugin:build报错

实践 Mac Docker

问题描述

在Mac下执行如下命令clone项目

$ git clone https://github.com/carnellj/spmia-chapter1

之后对该项目执行命令构造docker镜像。

mvn docker:build

结果产生如下错误

解决办法

查询https://github.com/spring-projects/spring-boot/issues/24638得知：默认情况下，Mac下的docker没有暴露tcp的socket。

所以需要在一个窗口执行如下命令：

# 暴露出tcp socket
socat TCP-L:2375,fork UNIX:/var/run/docker.sock

之后打开一个新窗口，首先执行如下命令：

# 指定docker下暴露的tcp socket信息
export DOCKER_HOST=tcp://localhost:2375

之后重新执行构造镜像的命令：

mvn docker:build
