Java中线程的状态及转换

Java 多线程 
public enum State {
		// 线程对象刚被创建，还没有调用start函数
    NEW,
		// 线程正在被JVM执行（包括等待其他资源的情况，例如IO操作。）
    RUNNABLE,
		// 线程正在等待获取一个锁
    BLOCKED,
		// 线程由于调用了下面的某个方法而处于等待状态。 wait、join、park
    WAITING,
		// 线程调用了下面的方法而处于限时等待状态。sleep、wait、join
    TIMED_WAITING,
		// 线程已经执行完毕
    TERMINATED;
}

如上，根据Java的Thread中对线程状态的枚举可知，Java中线程状态被分为6种，下图描述了各个状态之间的转化
