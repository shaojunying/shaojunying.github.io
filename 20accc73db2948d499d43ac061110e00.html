Java中的i++与++i

JVM

对于如下两个命令

int i = 1;
i = i ++;

和

int i = 1;
i = ++i;

i = i ++对应的字节码为

iload_1 将局部变量表中下标为1的元素压入栈顶
iinc1_1 将局部变量表中下标为1的元素加一
istore_1 将栈顶元素存入局部变量表中下标为1的位置

可以看到，上面局部变量表中的值加一之后将被覆盖掉，最后i的值还是1

i = ++i对应的字节码为

iinc_1
iload_1
istore_1

最终i的值被加一，最后i的值为2
