M1 Mac下python安装pandas

实践 Python

使用pip安装

执行如下命令即可通过pip安装pandas

pip3 install pandas

安装过程中的截图如下所示。可能会在Getting requirements to build wheel ... 和Building wheel for pandas (pyproject.toml) .. 卡顿一会儿。不过稍等一段时间即可安装成功。

之前以为卡顿那么久，应该是网络连接不上。结果后来看了一下竟然成功安装了

pip安装本地pandas包

  • 使用git clone git@github.com:pandas-dev/pandas.git 下载代码
  • cd pandas
  • 执行pip3 install . 即可安装。

本地编译并安装pandas库

  • 使用git clone git@github.com:pandas-dev/pandas.git 下载代码
  • cd pandas
  • 执行python3 setup.py instal 即可编译并安装。（如果安装过程中提示权限不够，前面添加一个sudo即可）

安装过程中可能会显示一些警告，忽略即可。

之前以为这些警告是报错，就直接退出了。最后又尝试了一下，发现这些警告不影响最终的成功安装。
